# 造作画布 ZZHB 0.1

造作画布是一款面向设计工作流的本地 AI 画布。你可以在同一个项目里整理参考图、提示词、模型结果、对话记录和设计说明，再把它们连接成可以持续修改的工作流程。

本仓库只用于发布 Windows 可执行版本和公开使用说明，源代码不公开。

## 下载与启动

1. 前往 [Releases](https://github.com/yangwc18181/zaozuohuabu-windows-preview/releases/latest)。
2. 下载 `ZZHB-0.1.0-win-x64.zip`。
3. 完整解压 ZIP，不要直接在压缩包里运行。
4. 双击 `ZZHB.exe`，等待本地服务启动并自动打开浏览器。

当前版本支持 Windows x64。由于尚未购买代码签名，Windows 可能显示“未知发布者”，请确认文件来自本仓库后再运行。

## 第一次使用

首次进入后，打开“模型与供应商”：

- 当前模型和供应商连接已经预设完成。
- 只需在准备使用的供应商处填写自己的 API Key，无需手动填写 Base URL、协议或模型编号。
- 软件包不包含开发者或其他用户的 Key。
- ComfyUI 工作流需要你的电脑已安装并启动对应的本地 ComfyUI 服务。

模型调用可能产生费用。使用任何官方或第三方供应商前，请先确认价格、额度、隐私政策和服务条款，建议先进行少量测试。

## 0.1 包含什么

- 25 条可选择的模型目录，以及已经关联好的供应商调用配置。
- 3 份 ComfyUI 工作流：白膜参考参考图、草图渲染、高清放大。
- 3 个首页官方模板：草图渲染、指示框改图、提示词反推。
- 普通文本节点和独立对话节点。
- 图片节点、便签节点、画布连接、项目保存与历史内容恢复。
- 模型预设、工作流和画布模板的本地导入导出能力。

## 本地数据与 API Key

- 项目、配置和媒体保存在 `%LOCALAPPDATA%\ZaoZuoCanvas`。
- API Key 由当前 Windows 用户的 Credential Manager 保存，不写入程序目录或发布 ZIP。
- 发布包不包含任何用户项目、项目素材、API Key、日志或缓存。
- 删除解压后的程序目录不会自动删除 `%LOCALAPPDATA%\ZaoZuoCanvas` 中的本地项目。

请不要在 GitHub Issues 中提交 API Key、客户资料、私人项目、未脱敏截图或完整日志。
