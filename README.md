# ZZHB Windows 预览版

这是 ZZHB 的 Windows 早期预览包，用来给大家先体验、测试和反馈问题。

## 为什么做这个画布

我自己是做设计出身，在使用各种 AI 工具的过程中，越来越明显地感觉到：现在的 AI 能力已经很强，但真正适合设计工作流的画布工具还不够理想。

很多工具要么偏聊天，要么偏单次生成，要么更像技术工作流编辑器。它们可以完成某个单点任务，但在设计过程中常见的这些操作：

* 收集参考图
* 写想法和提示词
* 对比不同方案
* 反复修改局部
* 保存生成历史
* 把图片、文字、模型结果放在同一个项目里整理

这些流程还没有一个特别顺手的承载方式。

所以我想做一个更适合设计类工作的 AI 画布。它不只面向室内设计，后续也希望能适用于更多设计场景，比如空间、产品、视觉、软装、概念方案、风格探索等。

目前这个版本还很早期，但我希望先把它放出来，让真实用户试一试，看看它到底在哪些地方有用，哪些地方不好用，哪些功能应该优先改。

## 当前状态

这个版本仍处于非常早期的预览阶段，不是稳定版，也不建议用于正式生产项目或重要客户项目。

这个工具是我作为非编程人员，在 AI 协作辅助下开发出来的。虽然已经做过一些基础测试和打包检查，但由于开发方式和当前阶段限制，软件中一定还会存在不少 bug、交互问题、兼容性问题或设计不完善的地方。

如果你愿意试用，欢迎把遇到的问题反馈出来。你的反馈会非常有帮助。

## 下载和运行

1. 在 Release 中下载 `ZZHB-v0.1.0-preview.zip`
2. 解压后运行 `ZZHB.exe`
3. 第一次进入后，先到「模型配置」里添加或选择可用模型
4. 画布里已经预设了几个模型模板。如果你有官方 API Key，可以直接配置官方模型；如果你不熟悉模型配置，也可以使用内置的第三方中转配置，通过中转服务入口注册账号并获取 Key 后试用
5. 配置好模型和 API Key 后，就可以在画布中测试文本、图片等节点式创作流程

目前预览版中主要接入了以下模型模板：

* Gemini 3.1 Flash Lite 文本模型
* GPT Image 2
* Nano Banana 2
* Nano Banana Pro

需要提醒的是：如果你使用第三方中转服务，前期建议先少量充值测试，确认模型可用性、速度、价格和稳定性都适合自己后，再根据需要继续使用。

后续版本会继续适配更多模型，包括视频模型和其他多模态模型，也会逐步增加更多节点和更完整的设计工作流能力。

## 关于 API Key 和本地数据

模型 API Key 会保存在你本机的预览包目录中，例如 `UserData` 相关目录。

请注意：

* 不要把自己的 `UserData` 文件夹发给别人
* 不要在 GitHub Issues 里提交 API Key
* 不要上传未脱敏的项目文件、设计资料、截图或日志
* 反馈问题时，尽量只描述操作步骤、错误现象和系统环境

## 关于内置中转模型配置

为了让不熟悉模型配置的用户也能快速开始体验，预览包中可能会内置一些第三方中转接口的模型配置示例。

这类配置的目的只是降低早期试用门槛。不同中转接口、不同模型、不同路径和参数格式之间差异很大，后续也可能发生变化。

你也可以自行配置 OpenAI、Gemini 或其他兼容接口。

需要注意的是：

* 第三方接口的稳定性可能会变化
* 第三方接口的价格、额度、服务条款和隐私政策由对应服务方决定
* 使用任何第三方接口前，请自行了解它的服务条款和风险
* 本项目提供的配置示例不代表官方背书或长期可用承诺

如果你有更稳定、更适合国内用户使用的模型中转资源，也欢迎通过 Issues 提建议。
